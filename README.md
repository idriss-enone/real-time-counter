# Real-Time Counter

Un compteur de texte réel; fluide et moderne, dévéloppé en technologies web native (HTML5,CSS3, JavaScript Vanilla).

---

## Fonctionnalités

- **Compteur de caractères:** Met à jour instantanément le nombre total de signes saisis.
- **Compteur de mots:** Calcul précisément le nombre de mots ignorant les espaces superflus. instantanément le nombre total de signes saisis.
- **Temps de lecture estimé:** Donne une estimation en minutes basée sur une vitesse moyenne de lecture(255 mots/min).
- **Interface responsive:** S'adapte parfaitement aux ordinateurs, tablettes et smartphones. 

---

## Technologies utilisées

- **HTML5**: Structuration sémantique de l'interface.
- **CSS3**: Design moderne, mise en page flexbox et animations fluides.
- **JavaScript (ES6+)**: Manipulation du DOM en temps réel via l'écouteur d'événements `input`.

---

## Installation et utilisation

Aucune installation complexe ou serveur n'est requis. Suivez simplement ces étapes:

1. **Cloner le dépôt** :

  ```bash
  git clone https://github.com/idriss-enone/real-time-counter.git 
  ```
2. **Accéder au dossier**

  ```bash
  cd real-time-counter
  ```

3. **Lancer l'application**

ouvrez le fichier `index.html` directement dans  votre navigateur web préféré.
